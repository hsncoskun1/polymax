## Changes

### `backend/app/services/market_fetcher.py`
- Added `enable_order_book: bool | None = None` field to `FetchedMarket`
- Added `tokens: list | None = None` field to `FetchedMarket`
- `_normalize()` now extracts both fields from the raw Gamma API dict
  - `enable_order_book`: `None` when absent; `bool(value)` otherwise
  - `tokens`: `None` when absent or not a list; `[]` when present but empty

### `backend/app/services/market_discovery.py`
- Added `NO_ORDER_BOOK = "no_order_book"` to `RejectionReason`
- Added `EMPTY_TOKENS = "empty_tokens"` to `RejectionReason`
- Rule order is now:
  1. INACTIVE (`active=False` or `closed=True`)
  2. NO_ORDER_BOOK (`enable_order_book is not True` — covers `False` and `None`)
  3. EMPTY_TOKENS (`not market.tokens` — covers `None` and `[]`)
  4. MISSING_DATES (`source_timestamp` or `end_date` is `None`)
  5. DURATION_OUT_OF_RANGE (outside [240, 360] s window)
